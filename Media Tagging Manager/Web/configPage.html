Select Libraries

Choose which Jellyfin libraries the plugin can access. Only selected libraries are read or changed, whatever you named the library. Within each selected library, Movies and Series are tagged; episodes inherit their series context.

Watchmode limits API requests by allowing 2,500 during cycles of 30 days. The first 30 day cycle starts the day you request the API key. On your Watchmode account, it will show you which date your requests renew, titled 'Quota Resets On.' Input that date here so this plugin can track your requests and automatically respect Watchmode's limits.
Enter the current usage shown in your Watchmode account for this 30-day cycle. For example, enter 106 if that is your account’s current total. Future Watchmode requests made by this plugin add to this number.

Logo Settings

This plugin uses provider and network logos by default. Logos are used in the pick providers and networks lists on the Network and Provider Settings tab, and on the View and Edit Tags tab in the Library Overview. Coming soon, additional plugins by the same developer will utilize logos for your detail pages and for collection making. To disable this plugin from saving (and using) logos, uncheck the option below and save your settings. Then click the 'Delete cached logos' button.

Each source logo is limited to 2 MB. The default total cache limit is 100 MB.

Logo cache status will load shortly.

Load All Logos loads Provider logos and preloads Network logos before any media scan whenever the source catalog supplies a TMDb network ID. A network remains without a logo only when the source has no matching TMDb ID or TMDb has no logo for it.

Newly Added Media Settings

Choose your preference for how this plugin will interact with newly added media in your selected libraries. When enabled, only new Movies and Series added after it was turned on are checked after a normal Jellyfin library scan. Turn it off to prevent automatic checks for incoming media; manual and scheduled full scans remain available.

Scheduled Tasks

Media often changes which streaming provider or network it is shown on. Enable a full-library scan at your selected interval to keep tags current. Also available as “Refresh provider and network tags” under Dashboard → Scheduled Tasks.

Example: if The Hunger Games was tagged Provider: Netflix and later moves to Amazon Prime—or is no longer hosted anywhere—turning this on removes the old plugin-created Netflix tag during the next scan. Leaving it off preserves the old plugin-created tag. Your other Jellyfin tags are never removed.

Availability Region Settings

Set up to three countries used for streaming availability. The choices are TMDb countries with watch-provider data.

Tag Settings

Choose if you want provider tags, network tags, or both. Existing Jellyfin tags created without this plugin are never removed. This plugin only adds new tags and only removes tags it created.

Adds tags such as Provider: Netflix for currently available streaming, rental, purchase, or free providers.

Adds tags such as Network: BBC One for a television series’ network.

TV Network Streaming Apps Settings

Some television networks have their own streaming apps, such as BBC iPlayer. Choose how titles available through those apps are tagged. TMDb and Watchmode can each return Network tags from actual title-level network metadata. This plugin never creates a Network tag from an app name alone. To use Both fully, select the streaming app in Select Providers and its network in Select Networks below.

Adds an API-returned tag such as Network: BBC One, without adding Provider: BBC iPlayer.

Adds Provider: BBC iPlayer, without adding a network tag for that title.

Adds both the streaming-app Provider tag and any API-returned Network tag.

Loading network options…

Save Network Selections applies your choices to future scans only. It does not scan media or change existing tags. Sync with Only Selected Networks is for removing existing unselected plugin Network tags.

Genre Settings

Choose which genres Media Tagging Manager can add to media in your selected libraries. Genres are provided by TMDb and are added as separate tags, such as Genre: Drama or Genre: Documentary. Existing Jellyfin genres and tags not added by this plugin are never removed.

Loading genre options…

Keywords Settings

Choose whether Media Tagging Manager adds keywords during scans of your selected libraries. Keywords are specific terms returned by TMDb for a title, such as Keyword: Time Travel, Keyword: Broadway, or Keyword: Based on Novel. Keywords are added only when this setting is enabled before a scan.

To remove keywords added by this plugin, uncheck the box and click the 'Remove Keywords Added by This Plugin' button.

Collections Tags Settings

Scan selected libraries for direct TMDb movie collection matches. Review results by library, choose the matches you want, and add only those Collection: tags. This uses TMDb collection membership only; it does not infer collection membership.
