Settings & sources

Libraries

Only selected libraries are read or changed. Movies and series are tagged; episodes inherit the series context.

Tag behavior

Your other Jellyfin tags are never removed. Turning off replacement preserves previously found plugin tags.

Two-letter country code, for example US, GB, CA, or AU.

Automatic refresh

Also available as “Refresh provider and network tags” under Dashboard → Scheduled Tasks.

When enabled, only new movies and series added after it was turned on are checked after a normal Jellyfin library scan. Turn it off to prevent automatic checks for incoming media; manual and scheduled full scans remain available.

Safety backups and undo

A backup saves the complete current tag list for every item in the selected libraries. Restoring a backup overwrites current tags for those saved items, including tags not created by this plugin.

Built-in sources

TMDb contributes region-specific watch providers and TV networks. Paste a Read Access Token, not an API key; leave blank to disable it.

This product uses TMDB and the TMDB APIs but is not endorsed, certified, or otherwise approved by TMDB. Learn more about TMDB.
