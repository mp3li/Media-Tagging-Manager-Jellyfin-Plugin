# 0.1.0.1-test catalog-install build

This folder contains the test ZIP used to validate Jellyfin repository installation before the first stable release.

- Target Jellyfin ABI: `10.11.3.0`
- Plugin version: `0.1.0.1`
- Purpose: test installation, settings, scans, backups, and uninstall/update behavior on a real Jellyfin server.

This is not a stable release. It contains no API credentials, Jellyfin configuration, backups, logs, or media data.
