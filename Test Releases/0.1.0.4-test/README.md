# 0.1.0.4-test catalog-install build

This folder contains the test ZIP compiled against the official Jellyfin
10.11.11 package references for the active test server.

- Target Jellyfin ABI: `10.11.11.0`
- Plugin version: `0.1.0.4`
- Purpose: test catalog installation, library selection, settings, backups,
  scans, and update behavior on a Jellyfin 10.11.11 server.

This is not a stable release. It contains no API credentials, Jellyfin
configuration, backups, logs, or media data.
